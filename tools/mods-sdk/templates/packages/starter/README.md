# Package Mod

This is a package mod that bundles multiple mods together.

## Sub-mods

| Folder           | Type          | Description                  |
| ---------------- | ------------- | ---------------------------- |
| `actions/`       | Action Mod    | An action mod.               |
| `visualizations/`| Visualization | A visualization mod.         |

## Prerequisites

- [Node.js](https://nodejs.org/) (v22 or later)
- [Visual Studio Code](https://code.visualstudio.com/) (recommended)

## Getting started

Install dependencies and build all sub-mods:

```bash
npm install
npm run build
```

To build in watch mode with debug output:

```bash
npm run build:dev
```
